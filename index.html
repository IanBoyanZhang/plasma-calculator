Plasma Parameters Calculator

This calculator calculates plasma parameters using NRL Plasma Formulary. Default input values are based on typical CSDX argon discharges.

Input Quantities

Magnetic Field (G)
Density (cm-3)
Electron Temperature (eV)
Ion Temperature (eV)
Ion Mass Number
Charge Number
Coulomb Logarithm

Frequencies

Elec coll. rate (sec-1)
Ion coll. rate (sec-1)
Elec gyro-freq (Hz)
Ion gyro-freq (Hz)
Elec plasma freq (Hz)
Ion plasma freq (Hz)

Lengths

Elec gyro-radius (cm)
Ion gyro-radius (cm)
Hybrid Ion gyro-radius (cm)
Elec skin depth (cm)
Ion skin depth (cm)
Debye length (cm)

Velocities

Elec thermal velocity (m/s)
Ion thermal velocity (m/s)
Ion sound speed (m/s)
Alfven velocity (m/s)
